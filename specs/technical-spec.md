## System Requirements

### Minimum Hardware

- **CPU:** 2 cores (x86_64)
- **RAM:** 4GB
- **Disk:** 5GB free space
- **Network:** Internet connection (Docker image pulls)

### Software Dependencies

- **OS:** Ubuntu 22.04+ / WSL2 / macOS / Windows with Docker Desktop
- **Docker:** Engine 24.x+
- **Docker Compose:** v2.x+
- **Rust:** 1.70+ (for building CLI)
- **Python:** 3.11+ (for faucet development)
